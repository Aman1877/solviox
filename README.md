# SOLVIOX Energy & Lighting Website

A modern, responsive website for SOLVIOX Energy & Lighting, a solar energy solutions company based in Ahmedabad, Gujarat.

## 🌟 Features

- **Modern Design**: Clean, professional design with smooth animations
- **Responsive Layout**: Optimized for all devices (mobile, tablet, desktop)
- **Smooth Animations**: Powered by Framer Motion for engaging user experience
- **Multilingual Support**: Gujarati and English content
- **Contact Forms**: Interactive contact forms with React Hook Form validation
- **EmailJS Integration**: Direct email sending from contact form
- **Service Showcase**: Detailed presentation of solar energy services
- **Performance Optimized**: Fast loading times and smooth interactions

## 🛠️ Tech Stack

- **React 18** - Modern React with hooks
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and development server
- **Framer Motion** - Smooth animations and transitions
- **React Hook Form** - Performant forms with easy validation
- **EmailJS** - Client-side email sending for contact forms
- **Lucide React** - Beautiful, customizable icons
- **Custom CSS** - Modern CSS with CSS variables and responsive design

## 🚀 Services Featured

1. **Solar Rooftop** - Residential and commercial rooftop solar solutions
2. **Solar LED & AC Street Light** - Energy-efficient street lighting
3. **Solar & AC EPC Projects** - End-to-end project management
4. **Solar & AC High Mast** - High-mast lighting solutions

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and visit `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## 📧 EmailJS Setup

The contact form uses EmailJS to send emails directly from the client-side. To enable this functionality:

1. **Follow the setup guide**: See `EMAILJS_SETUP.md` for detailed instructions
2. **Create EmailJS account**: Sign up at [emailjs.com](https://www.emailjs.com/)
3. **Configure email service**: Set up Gmail or other email provider
4. **Create email template**: Use the provided template structure
5. **Update credentials**: Replace placeholder values in `src/utils/formSubmission.ts`

**Note**: Until EmailJS is configured, the form will show a placeholder message.
